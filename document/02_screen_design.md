## 2. 各画面詳細

### S-01: トップページ

**目的**: ゲームの入り口。ルーム作成のみ。

```
┌────────────────────────────────────┐
│                                    │
│         🎮 ファイブリーグ          │
│                                    │
│    5人で挑む文字当てクイズ！       │
│                                    │
│   ┌──────────────────────────┐    │
│   │    ルームを作成する      │    │
│   └──────────────────────────┘    │
│                                    │
│                                    │
└────────────────────────────────────┘
```

**要素**:
- タイトルロゴ
- キャッチコピー
- 「ルームを作成する」ボタン

---

### S-02: ニックネーム入力

**目的**: プレイヤーの表示名を設定。

```
┌────────────────────────────────────┐
│                                    │
│         ニックネームを入力         │
│                                    │
│   ┌──────────────────────────┐    │
│   │                          │    │
│   └──────────────────────────┘    │
│         最大10文字                 │
│                                    │
│   ┌──────────────────────────┐    │
│   │       入室する           │    │
│   └──────────────────────────┘    │
│                                    │
└────────────────────────────────────┘
```

**要素**:
- ニックネーム入力フィールド（最大10文字）
- 「入室する」ボタン
- バリデーション: 空欄不可、10文字以内

---

### S-03: 待機ルーム

**目的**: プレイヤー集合、順番設定、ゲーム開始。

```
┌────────────────────────────────────┐
│  ルームID: ABC123                  │
│  ┌────────────────────────────┐   │
│  │ 🔗 リンクをコピー          │   │
│  └────────────────────────────┘   │
├────────────────────────────────────┤
│                                    │
│  参加者 (3/5)                      │
│                                    │
│  ┌─┐ ┌─┐ ┌─┐ ┌─┐ ┌─┐            │
│  │1│ │2│ │3│ │4│ │5│            │
│  └─┘ └─┘ └─┘ └─┘ └─┘            │
│  太郎 花子 次郎  --   --          │
│  (host)                            │
│                                    │
│  ─────────────────────────────    │
│  ※ホストのみ: ドラッグで順番変更  │
│  ─────────────────────────────    │
│                                    │
│   ┌──────────────────────────┐    │
│   │   ゲームスタート (3/5)   │    │ ← 5人未満は非活性
│   └──────────────────────────┘    │
│                                    │
│   [キック] ← ホストのみ表示       │
└────────────────────────────────────┘
```

**要素**:
- ルームID表示
- 共有リンクコピーボタン
- 参加者リスト（1〜5の番号付き）
- ホスト: ドラッグ&ドロップで順番変更可能
- ホスト: 「ゲームスタート」ボタン（5人揃ったら活性化）
- ホスト: プレイヤーキックボタン
- 一般: 待機メッセージ表示

### S-04: クイズ出題

**目的**: 問題を全員に表示。

```
┌────────────────────────────────────┐
│  Q.1          正解数: 0           │
├────────────────────────────────────┤
│                                    │
│  南アジアに位置する国、            │
│  ブータンの首都は？                │
│                                    │
│   ┌───┬───┬───┬───┬───┐         │
│   │ 1 │ 2 │ 3 │ 4 │ 5 │         │
│   ├───┼───┼───┼───┼───┤         │
│   │   │   │   │   │   │         │
│   └───┴───┴───┴───┴───┘         │
│   太郎 花子 次郎 四郎 五郎        │
│         ↑                         │
│      あなた                        │
│                                    │
│                                    │
│   ※ 数秒後に回答画面へ            │
└────────────────────────────────────┘
```

**要素**:
- 問題番号
- チーム正解数
- 問題文
- 5つの回答欄（空欄）
- 各欄に担当プレイヤー名
- 自分の担当位置をハイライト
- 自動遷移（3秒後に回答入力へ）

---

### S-05: 回答入力

**目的**: 手書きで1文字を入力。

```
┌────────────────────────────────────┐
│  あなたは 2文字目 を担当           │
├────────────────────────────────────┤
│                                    │
│  問題: ブータンの首都は？          │
│                                    │
│   ┌───┬───┬───┬───┬───┐         │
│   │ ? │ ? │ ? │ ? │ ? │         │
│   └───┴───┴───┴───┴───┘         │
│         ↑                         │
│                                    │
│  ┌────────────────────────────┐   │
│  │                            │   │
│  │                            │   │
│  │    (手書きキャンバス)      │   │
│  │                            │   │
│  │                            │   │
│  └────────────────────────────┘   │
│                                    │
│   [クリア]      [提出する]         │
│                                    │
└────────────────────────────────────┘
```

**要素**:
- 担当文字番号の表示
- 問題文（短縮版）
- 回答欄プレビュー（自分の位置をハイライト）
- 手書きキャンバス（Canvas）
  - タッチ/マウス対応
  - 線の太さ: 固定
  - 色: 黒固定
- 「クリア」ボタン（書き直し）
- 「提出する」ボタン

**キャンバス仕様**:
- サイズ: 300x300px（レスポンシブ調整）
- 背景: 白
- 線色: 黒
- 線幅: 8px
- 出力: PNG形式（Base64）

---

### S-06: 回答待機

**目的**: 他プレイヤーの提出を待つ。

```
┌────────────────────────────────────┐
│  回答を待っています...             │
├────────────────────────────────────┤
│                                    │
│   ┌───┬───┬───┬───┬───┐         │
│   │ ✓ │ ✓ │   │ ✓ │   │         │
│   └───┴───┴───┴───┴───┘         │
│   太郎 花子 次郎 四郎 五郎        │
│   提出  提出 入力中 提出 入力中   │
│                                    │
│         (3/5) 提出済み             │
│                                    │
│   ┌────────────────────────┐      │
│   │ ● ● ○ ● ○              │      │  ← アニメーション
│   └────────────────────────┘      │
│                                    │
└────────────────────────────────────┘
```

**要素**:
- 待機メッセージ
- 各プレイヤーの提出状況
  - ✓: 提出済み
  - 空欄: 入力中
- 提出人数カウント
- ローディングアニメーション

### S-07: 結果表示

**目的**: 全員の回答を確認し、各自が正誤を判定。

```
┌────────────────────────────────────┐
│  Q.1 結果発表！      正解数: 0    │
├────────────────────────────────────┤
│                                    │
│  問題: ブータンの首都は？          │
│  正解: テ ィ ン ブ ー              │
│                                    │
│   ┌───┬───┬───┬───┬───┐         │
│   │🖼️│🖼️│🖼️│🖼️│🖼️│  ← 手書き画像 │
│   └───┴───┴───┴───┴───┘         │
│   太郎 花子 次郎 四郎 五郎        │
│                                    │
│  ─────────────────────────────    │
│  あなたの判定:                     │
│                                    │
│   ┌─────────┐   ┌─────────┐       │
│   │  正解   │   │  不正解  │       │
│   │  (赤)   │   │  (青)    │       │
│   └─────────┘   └─────────┘       │
│                                    │
│  判定状況: 3/5 完了                │
│                                    │
│  ─────────────────────────────    │
│   [次の問題へ]  [ゲーム終了]      │  ← ホストのみ
└────────────────────────────────────┘
```

**要素**:
- 問題番号
- 問題文
- 正解（5文字を1文字ずつ表示）
- 全員の手書き回答（画像）
- 各回答の下にプレイヤー名
- 判定済みの回答には色付き枠（赤=正解、青=不正解）
- 自分の判定ボタン（正解/不正解）
- 判定完了人数
- ホスト: 「次の問題へ」ボタン
- ホスト: 「ゲーム終了」ボタン

## 4. レスポンシブ対応

### ブレークポイント

| デバイス | 幅 |
|----------|-----|
| スマートフォン | 〜640px |
| タブレット | 641px〜1024px |
| PC | 1025px〜 |

### スマートフォン対応の注意点

- 手書きキャンバスは画面幅の80%程度
- 5つの回答欄は横並び（小さめ）
- ボタンは十分なタップ領域確保（最低44x44px）
- スクロール可能なレイアウト
